# 記帳 App

## 簡介

一個自用的記帳 App。

## 架構

### 頁面

- 首頁
- 登入
- 註冊
- 收支記錄
- 收支分析
- 個人設定

### 記錄類型

#### 收入

- 薪資
- 獎金
- 其他

#### 支出

- 飲食
- 交通
- 購物
- 娛樂
- 旅遊
- 醫療
- 保險
- 學費
- 房租
- 網路費
- 手機費
- 運動
- 其他

## 技術環境

### 前端

- Next.js

### 後端

- Node.js
- Express.js
- Mongoose

### 資料庫

- MongoDB

### 其他

- React icons
- JWT
- @ducanh2912/next-pwa

## 指令

```bash
npm run dev
npm run build
npm run start
```
